## What's BetterDiscordPanel?

<a href="https://github.com/betterdiscordpanel" target="_blank">BetterDiscordPanel</a> is a new panel that allows you to message, manage and gain statistics inside of your discord bot.
<br/>

<!-- Features -->

## ‚ú® Features

- üìä **Discord Bot Statistics:** Gain full insights and details of your discord bot with server and bot information.
- üí¨ **Simple Messaging Interface:** BetterDiscordPanel allows you to easily message users and chat within servers using your discord bot. The panel also has a light and dark theme to suit your needs!
- üîí **Security:** BetterDiscordPanel does not run on any servers and can be self hosted. All bot details are stored locally and is completely secure.
- üëÄ **Opensource:** BetterDiscordPanel is open source on GitHub so you can see all the code behind the project.

<br/>

<!-- Getting Started -->

## üõ† Getting Started

This project does not run on any servers and can be self hosted. Simply open
`index.html` to get started or use the control panel by opening `BetterDiscordPanel.cmd` and executing "W" for the web panel.
<br/>

<!-- Installation -->

### **Step 2:** Setup / Installation

1. Clone the repository

```sh
git clone https://github.com/sanjaysunil/betterdiscordpanel
```

> ‚ö†Ô∏è **NOTE**: The updater only works if you have cloned the repository. 

2. Open `login.html` or open `BetterDiscordPanel.cmd` to run the panel.

3. Login to your Discord Bot.

4. Start using BetterDiscordPanel!
